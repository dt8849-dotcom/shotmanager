현재 데이터 요약
📤 백업 (데이터 내보내기)
JSON 파일로 저장됩니다. 안전한 곳(드라이브, 클라우드 등)에 보관하세요.
📥 복원 (데이터 불러오기)
이전에 백업한 JSON 파일을 선택하면 데이터가 복원됩니다. ⚠️ 현재 데이터는 덮어씌워집니다 — 먼저 백업하세요.
📋 앱 업데이트 시 데이터 유지 방법
  1. 이 화면에서 지금 백업하기로 파일 저장
  2. GitHub에 새 파일 업로드 (코드 업데이트)
  3. 업데이트된 페이지 새로고침
  4. 다시 이 화면에서 파일 선택으로 방금 저장한 백업 불러오기
💡 새 버전에서 추가된 기능들(새 필드, 새 탭 등)은 불러온 후 자동으로 채워집니다.
